SELF-FILLING FORM WITH DATA EXTRACTED FROM A PICTURE OF AN IDENTITY CARD BY COMPUTER VISION


Project description :

This project was part of my summer internship. The main objective of the project was to help the user fill a form effortlessly. He will only take a picture of his identity card and the system will do the job for him. Without needing to switch to arabic keyboard or focus to copy the long ID number. it consisted of the detection a the presence of a tunisian identity card in a picture. If positive, it proceeds to the extraction of the data ( first name, last name, ID number ... ). How I proceeded :
  • Scrapped data from the web, to form a dataset of dummy tunisian identity cards. Then preprocessed the data and labelled it.
  • Built a Tensorflow object detection model to detect the identity card.
  • Retrained an OCR model on custom arabic font which is the font of tunisian identity cards.
  • Joined the two models in a Flask application.
  • Deployed the application on AWS EC2 instance as a microservice.
  • Integrated and used the microservices in forms to become self-filling forms.


What I learnt :

✓ Be independent and autonomous.
✓ Put and use my machine learning models in production.
✓ Manage AWS instances.

VERIFICATION OF FACE IN SELFIE AND IDENTITY CARD


Project description :

I did this project in order to be integrated in a platform for online payment and bank account creation. In fact, the user is asked to take a picture of his identity card and a selfie. This system proceeds in doing the face verification of the two pictures and tell if it is the same person or not. What I did is :
  • Use of the Multi-Task Cascaded Convolutional Neural Network ( MTCNN ) to find and extract faces from both the selfie and the identity card.
  • Use of VGGFace2 for face verification.
As you can see in the demo above, first I tested on my own selfie and identity card, the result was True with a rate of similarity equal to 100%, then I tried with someone else's selfie and the result was False.
I deployed the application in an AWS EC2 instance and I integrated it in the online payment and bank account creation platform.

RESEARCH STUDY ABOUT IMAGES COMPRESSION USING GRAPH NEURAL NETWORKS (GNN)


Project description :

This project was my end of year project. It is a research project that aims to find and develop new images compression methods based on deep learning and especially the Graphic Neural Networks rather than the traditional methods. This project initiated me to the world of researches and helped me to further my analytical skills. We found, my team and I, a method of compression using the graph neural network that operates on a certain type of images which are the Light Field Images and we managed to generalize that method on other images types. The methods consists of the separation of the high frequencies and the low frequencies of the pictures then the sampling and training of the GNN upon the low frequencies. Finally we interpreted the quantitative and qualitative results. We were supervised by our professor who is a researcher in the field of image processing and we learned so much from her.


What I learnt :

✓ Explore scientific papers and write new ones.
✓ Think outside the box.
✓ Collaborate with researchers all around the world and ask for help when needed.
